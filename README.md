# 🧠 AI Examiner 2.0

AI Examiner 2.0 is an AI-powered automatic answer sheet evaluation system built with FastAPI and React.

It allows uploading:
- 📄 Question Paper
- 📊 Marking Scheme
- 📝 Student Answer Sheet

The system automatically evaluates answers and generates:
- ✅ Total Marks
- 📈 Percentage
- 🏆 Grade
- 🎓 Roll Number Based Result

---

## 🚀 Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Framer Motion
- Recharts

Backend:
- FastAPI
- Python 3.10
- SQLAlchemy
- JWT Authentication
- SQLite / PostgreSQL

---

## ⚡ Features

- Secure Login System
- Multi-PDF Upload
- AI-Based Evaluation
- Automatic Marks Calculation
- Subject-Based Results
- Public Result Access
- Admin Dashboard
