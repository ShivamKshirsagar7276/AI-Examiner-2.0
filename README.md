# 🧠 AI Examiner 2.0

AI Examiner 2.0 is an AI-powered automatic answer sheet evaluation system built with FastAPI and React.

It allows uploading:
- 📄 Question Paper  
- 📊 Marking Scheme  
- 📝 Student Answer Sheet  

The system automatically evaluates answers and generates:
- ✅ Total Obtained Marks  
- 📈 Percentage  
- 🏆 Grade  
- 🎓 Roll Number-Based Result  

---

## 🚀 Tech Stack

### 🌐 Frontend
- React.js  
- Tailwind CSS  
- Framer Motion  
- Recharts  

### ⚙ Backend
- FastAPI  
- Python 3.10  
- SQLAlchemy  
- JWT Authentication  
- SQLite / PostgreSQL  

---

## ⚡ Key Features

- 🔐 Secure Authentication System  
- 📤 Multi-PDF Upload Support  
- 🤖 AI-Based Answer Evaluation  
- 📊 Automatic Marks & Grade Calculation  
- 🎯 Subject-Based Result Management  
- 🌍 Public Result Access via Roll Number  
- 📈 Interactive Admin Dashboard  
- 🖥 Desktop App + Web Version Support
